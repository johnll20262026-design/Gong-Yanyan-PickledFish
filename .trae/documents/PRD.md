# 龚艳艳·老江津酸菜鱼餐厅宣传网站 PRD

## 1. 产品概述
为龚艳艳·老江津酸菜鱼餐厅打造一个高品质的宣传展示网站，展示餐厅特色、团购优惠和顾客好评，吸引顾客到店消费。
- 目标用户：武汉黄家湖周边居民、大学生、美食爱好者
- 市场价值：提升餐厅品牌形象，通过网络渠道扩大知名度，促进团购转化

## 2. 核心功能

### 2.1 页面模块
1. **首页（单页应用）**：英雄横幅、餐厅介绍、荣誉资质、团购套餐、顾客评价、店铺信息、联系导航

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | Hero英雄区 | 餐厅名称展示、特色标语、人均消费、主打菜品 |
| 首页 | 餐厅简介 | 餐厅故事、菜品特色、用餐环境介绍 |
| 首页 | 荣誉展示 | 金牌好店、武汉酸菜鱼好评榜、黄家湖美食榜第四名等荣誉徽章 |
| 首页 | 团购套餐 | 2-3个高销量团购套餐详细介绍，含价格、优惠力度、套餐内容 |
| 首页 | 顾客评价 | 精选优质顾客好评，含评分、评价内容 |
| 首页 | 店铺信息 | 地址、营业时间、人均消费、联系电话 |
| 首页 | 导航引导 | 地图位置引导、抖音链接入口 |
| 首页 | 技术支持按钮 | 右下角悬浮按钮，点击跳转到技术支持网站 |

## 3. 核心流程

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：翠绿色 (#12a17d)，参考技术支持网站色系，代表新鲜、健康
- **辅助色**：暖金色 (#d4a853) 用于强调价格、优惠标签；米白色 (#fefcf8) 背景
- **文字色**：深灰 (#1a1a1a) 正文，中灰 (#666) 辅助文字
- **按钮样式**：圆角按钮，主按钮绿色填充+白色文字，hover时轻微放大(1.05)和阴影
- **字体选择**：标题使用有设计感的中文字体，正文使用清晰易读的无衬线字体
- **布局风格**：卡片式布局，适当留白，高端精致的餐饮风格
- **图标风格**：简洁线性图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 首页 | Hero区 | 酸菜鱼美食大图背景，餐厅名称金色大字，人均消费徽章，"入围武汉酸菜鱼好评榜"标签 |
| 首页 | 餐厅简介 | 左右分栏，美食图片+文字介绍，讲述老江津酸菜鱼的传承故事 |
| 首页 | 荣誉展示 | 金色/绿色徽章式卡片，三个荣誉并列展示，带图标装饰 |
| 首页 | 团购套餐 | 大卡片设计，套餐图片+原价划线+现价高亮+购买按钮，销量标签 |
| 首页 | 顾客评价 | 头像+星级评分卡片，左右滑动或网格布局 |
| 首页 | 店铺信息 | 地址卡片、营业时间卡片、人均消费卡片、联系按钮 |
| 首页 | 技术支持按钮 | 右下角固定圆形绿色按钮，白色图标，hover放大效果，点击新窗口打开 |

### 4.3 响应式设计
- 桌面优先设计，适配1920px/1440px/1024px
- 平板适配（768px）：调整布局为单列
- 手机适配（375px）：优化文字大小、按钮尺寸，团购套餐竖向排列
- 触摸优化：按钮点击区域≥44px
